# קו פתוח

רשת קריאות לנהגים ולקוחות.

## משפט מוצר

פותחים קריאה. נהגים מגיבים. נוסעים באמון.

## מה זה

קו פתוח היא תשתית קריאות לנסיעות.

הלקוח לא "מזמין מונית".
הוא פותח קריאה.

הנהג לא "מחכה לאלגוריתם".
הוא נכנס למרחב קריאות, מגיב, מציע, נבחר ובונה אמון.
