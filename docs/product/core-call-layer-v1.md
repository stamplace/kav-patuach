# קו פתוח — Core Call Layer V1

## עיקרון

קריאה אחת מפעילה את כל המערכת.

## מה נוסף

נוספה שכבת קריאה חיה שמשתקפת בכל הטאבים:

- לקוח: הקריאה שלי
- נהג: קריאה לעבודה
- מרחב: אזור חי
- ניהול: שליטה
- אמון: ביטחון ואימות

## למה זה חשוב

הדמו מפסיק להרגיש כמו חמש תצוגות נפרדות.
הוא מתחיל להרגיש כמו מערכת אחת שבה פעולה אחת משנה את כל המרחב.

## גבול אמת

זה עדיין דמו מוצרי.
הקריאה החיה היא state פנימי בדפדפן, לא שרת אמיתי.

בשלב הבא אפשר לחבר:
- שרת
- DB
- מיקום
- נהגים אמיתיים
- לקוחות אמיתיים
- התראות
