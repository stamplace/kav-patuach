# קו פתוח — React Vercel Revival V1

## מטרה

להחיות את React/Vercel הקיים של קו פתוח ולהפוך אותו לדמו מכירתי פרימיום.

## עיקרון

לא מתחילים React חדש.
משתמשים בענף הארכיון של Next ומעבירים אליו את מה שהבשיל ב־Flutter:

- נכסי תמונה
- יום/לילה
- טאבים
- קריאה חיה
- שפת מוצר
- אינטראקציה מכירתית

## גבול אמת

זו גרסת React/Vercel דמו.
Flutter נשאר על main כגרסת אפליקציה עובדת ב־GitHub Pages.
