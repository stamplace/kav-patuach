# קו פתוח — Interface Plan

## 1. Landing

מטרה:
להציג את המוצר, השפה והכניסה לכל נקודת קצה.

צריך להציג:
- קו פתוח
- רשת קריאות לנהגים ולקוחות
- פתח קריאה
- נהגים על הקו
- אמון 98%
- כניסה ללקוח / נהג / מנהל

## 2. Customer

מטרה:
לאפשר ללקוח לפתוח קריאה.

פעולות:
- מאיפה אוספים אותך
- יעד הנסיעה
- עכשיו / מאוחר יותר
- פתח קריאה
- צפייה בתגובות נהגים

## 3. Driver

מטרה:
לאפשר לנהג לעבוד עצמאית.

פעולות:
- אני על הקו
- צפייה בקריאות באזור
- קבלת קריאה
- שליחת הצעה
- צפייה באמון ובהכנסה

## 4. Admin

מטרה:
בקרה ואמון.

פעולות:
- אישור נהגים
- צפייה בקריאות פעילות
- דיווחים לבדיקה
- אזורים חמים
- פעילות היום
