# קו פתוח — Pilot Definition of Done

## הפיילוט מוכן להצגה כאשר

- יש דמו חי וזמין בקישור
- האפליקציה נפתחת במצב יום
- ניתן לעבור למצב לילה
- טאב לקוח ברור
- טאב נהג ברור
- טאב מרחב ברור
- טאב ניהול ברור
- טאב אמון ברור
- הקריאה החיה משתקפת בכמה מסכים
- כפתורים מרכזיים מגיבים
- אין טקסטים שמחלישים את המכירה
- יש מסמך Scope
- יש גבולות ברורים ל־Production
- יש Roadmap לשלב הבא

## לא נחשב חסר בשלב הפיילוט

- אין אלפי משתמשים
- אין שרת Production מלא
- אין אפליקציות בחנויות
- אין סליקה
- אין מוקד שירות
- אין SLA
- אין כיסוי משפטי / ביטוחי

## משפט מכירה

זה לא סוף המוצר.
זה השלב שמראה שהמוצר חי, ברור, ומוכן להמשך השקעה.
