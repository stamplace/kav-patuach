# קו פתוח — Demo Activation Map V1

## מטרה

להפוך את הדמו ממסכים יפים למוצר שמרגיש חי בהצגה ללקוח.

## החלטה

לא מחברים backend בשלב הזה.
מחברים שכבת דמו אינטראקטיבית:
- פידבק לכל כפתור מרכזי
- פידבק לשדות בחירה
- פתיחה במצב יום
- השארת מצב לילה כאפשרות
- שמירה על מסלול GitHub Pages הרשמי

## טוויסט הדמו

Demo Action Sheet:
כל פעולה חשובה פותחת חלונית פרימיום שמסבירה מה קרה:
- הקריאה נפתחה
- הנהג קיבל קריאה
- הצעה נבחרה
- כתובת נבחרת

זה לא backend אמיתי, אבל זה נותן ללקוח תחושה של מוצר פעיל.

## גבול אמת

כל פעולה בדמו היא סימולציה.
בשלב הבא מחברים:
- שרת
- מסד נתונים
- הזדהות
- קריאות אמיתיות
- נהגים/לקוחות אמיתיים
