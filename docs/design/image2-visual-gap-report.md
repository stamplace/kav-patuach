# קו פתוח — Image2 Visual Gap Report

## השוואה אחד מול אחד

### רקע
באתר היה גריד ירוק נקי. בתמונות Image2 יש עיר לילה, כבישים, עומק, אור, Skyline ונקודות רשת.
תיקון: CityBackdrop קיבל שכבות עיר, Glow, Skyline, גריד פרספקטיבי ורשת נקודות.

### מפה
באתר הייתה מפה שטוחה. בתמונות Image2 יש תחושת מפה חיה עם כבישים, חוף, Route Glow ופינים.
תיקון: LiveMapPainter קיבל Coast, כבישים, Route Glow רחב, פינים ותוויות בסגנון נאון.

### חומר
באתר הכרטיסים היו נקיים מדי. בתמונות Image2 יש זכוכית כהה, גבולות, צל ונפח.
תיקון: GlassPanel עבר לגרדיאנט כהה, גבול בהיר יותר וצל כפול.

### מצב יום/לילה
בתמונות הרפרנס שפת הלילה היא הראשית.
תיקון: נוסף VisualMode עם מתג יום/לילה בטופ־בר.

## כלל להמשך
כל מסך חדש חייב להיבדק מול:
- עומק רקע
- מקור אור
- Glow סביב פעולה ראשית
- מעט טקסט
- כפתור ראשי אחד
- אמון גלוי
