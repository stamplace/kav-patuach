# קו פתוח — Visual Asset Control Pass 01

## מטרה

לייצב את השפה אחרי הכנסת תמונות הפרימיום.

## בוצע

- מצב יום משתמש ב־kav-day-city.webp.
- מצב לילה משתמש ב־kav-night-city.webp.
- מסך אמון משתמש בתמונה השישית: kav-hero-poster-night.webp.
- מפות משתמשות ב־kav-map-dark.webp או kav-map-light.webp לפי מצב יום/לילה.
- שכבות CustomPainter ישנות הוחלשו כדי שלא יכערו את תמונת המפה.
- ה־Glow הוחלש כדי שלא יציף את כל הממשק בירוק.
- נוסף VisualModeScope כדי שכל רכיבי המפה ידעו אם הממשק במצב יום או לילה.

## Use Cases ברורים

לקוח:
פתיחת קריאה, איסוף, יעד, זמן נסיעה.

נהג:
אני על הקו, קריאות זמינות, הכנסה ואמון.

מרחב:
אזור חי, נהגים פעילים, קריאה חדשה.

ניהול:
לוח בקרה, מדדים, דיווחים, אישורים.

אמון:
שכבת אמון, אימות, מסמכים, דירוגים, נסיעות.
