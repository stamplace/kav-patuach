# קו פתוח — Use Case Pass 02: Driver

## מטרה

להפוך את טאב נהג ממסך דמו למסך עבודה ברור.

## Use Case

נהג עובד:
1. נכנס למצב על הקו
2. רואה מפה וקריאות קרובות
3. רואה הכנסה, אמון וכמות קריאות
4. מקבל קריאה מומלצת
5. רואה תור קריאות נוסף

## בוצע

- Header פעולה ברור.
- כרטיס זמינות LIVE.
- מדדי עבודה.
- קריאה מומלצת עם סכום ופעולת קבל.
- רשימת קריאות קצרה.
- מפה נשארת מרכז חזותי אבל לא בולעת את כל המסך.

## בדיקה

בטאב נהג צריך להיות ברור תוך שנייה:
הנהג זמין, יש קריאות, ויש פעולה ברורה לקבל.
