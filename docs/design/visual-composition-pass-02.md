# קו פתוח — Visual Composition Pass 02

## מטרה

לתקן את הבעיה שבה תמונות הפרימיום נכנסו, אבל שכבות ה־UI הישנות עדיין לכלכו את המסך.

## בוצע

- מסך אמון משתמש תמיד ב־kav-hero-poster-night.webp.
- מצב יום/לילה מכריח רענון של הסצנה הפעילה.
- טקסטורת המפה האמיתית מובילה; ציור הקוד הישן כמעט כבוי.
- הצבע הירוק רוכך לצבע פרימיום פחות פלסטיקי.
- הפאנלים הוחשכו והפכו נקיים יותר כדי שלא יראו עיר מלוכלכת מאחור.
- שכבות גריד/Glow ישנות הוחלשו.

## בדיקה

- לעבור בין יום/לילה.
- לפתוח אמון ולראות את התמונה השישית.
- לפתוח לקוח/נהג/מרחב ולוודא שהמפה נראית תמונה ולא גריד.
- לבדוק שהירוק פחות צורם.
