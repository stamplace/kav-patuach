# קו פתוח — Use Case Pass 03: Zone

## מטרה
להפוך את טאב מרחב למסך תפעולי ברור.

## Use Case
מפעיל/נהג רואה אזור חי:
1. כמה נהגים זמינים
2. כמה קריאות פתוחות
3. איפה העומס
4. האם צריך לנתב נהגים
5. מה האירועים החשובים עכשיו

## בדיקה
בטאב מרחב צריך להיות ברור תוך שנייה:
זה אזור חי, יש עומס, יש נהגים, ויש פעולה תפעולית.
