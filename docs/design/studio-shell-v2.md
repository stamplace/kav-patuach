# קו פתוח — Studio Shell V2

סטטוס:
החלפה של שכבת התצוגה הראשית לשפת מוצר פרימיום חדשה.

עקרונות:
- לא פוסטר.
- לא משחק נאון.
- לא בלוקים ענקיים.
- מוצר עבודה חי, יוקרתי, מדויק.
- מובייל קודם.
- לילה ברירת מחדל.
- יום קיים, אבל שקט ומבוקר.

מה השתנה:
- הבית הראשי עבר ל־StudioHome.
- כל הטאבים נבנים דרך StudioScene אחיד.
- Header, Hero, Cards, Maps, CTA ו־BottomNav קיבלו שפה אחת.
- הסצנות הישנות נשארו בקוד כרפרנס בלבד.
