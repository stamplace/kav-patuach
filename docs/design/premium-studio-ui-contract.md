# קו פתוח — Premium Studio UI Contract

## כיוון עיצובי

קו פתוח צריך להיראות כמו מוצר תחבורה פרימיום־הייטק:
נקי, דק, מדויק, שקט, יוקרתי, חי.

לא:
- פוסטר ענק
- משחק נאון
- טקסטים גדולים מדי
- קופסאות כבדות
- עומס גרפי

כן:
- ממשק כלי עבודה חי
- טיפוגרפיה קטנה ומדויקת
- כרטיסים דקים
- גבולות 1px
- עומק עדין
- CTA ברור
- מובייל קודם

## Typography

Mobile:
- H1: 28–34px
- H2: 20–22px
- Section label: 12–13px
- Body: 14–15px
- Microcopy: 11–12px

Weights:
- Titles: 800–900
- Labels: 700–800
- Body: 500–600

Line height:
- Titles: 1.05–1.15
- Body: 1.35–1.45

## Spacing

Mobile:
- Page padding: 16px
- Card padding: 14–18px
- Section gap: 14–20px
- Component gap: 8–12px

No oversized vertical gaps.

## Radius

- Buttons: 8–14px
- Cards: 14–20px
- App shell: 24–32px
- No huge pill shapes unless CTA requires it.

## Color System

Primary dark:
- #050B12
- #08121D
- #0B1724

Text:
- Primary: #F4F7FA
- Secondary: #AEB8C3
- Muted: #6F7A86

Accent:
- Emerald: #19D98B
- Mint: #6EF2BD
- Gold: #E8B95E

Rules:
- One primary accent per screen.
- Gold only for trust/premium.
- Green is action/status, not decoration everywhere.

## Components

### App Bar
- Slim
- Brand right
- CTA left
- Optional mode icon
- Height around 56–64px

### Cards
- Dark glass
- 1px border
- Low opacity glow
- No heavy neon borders

### Buttons
- Small/medium
- Strong text
- Subtle gradient
- Hover/tap feedback only

### Maps
- Must feel functional, not decorative.
- Route and pins are clear.
- Labels small.

### Bottom Nav
- Compact
- 5 tabs max
- Active state clear but not oversized

## Scenes

Each scene must include:
- One title
- One primary action or status
- One functional center visual
- One trust/status cue
- No marketing paragraphs

## Definition of Done

A screen is ready only if:
- It looks like a product, not a poster
- It can be used on mobile
- It has no oversized text
- It has no unnecessary blocks
- The primary action is obvious
- Visual language is consistent across all scenes
