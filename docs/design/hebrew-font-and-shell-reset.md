# קו פתוח — Hebrew Font + Shell Reset

## החלטה

Studio Shell V2 היה נקי מדי ופשוט מדי.
הוא הוריד את המוצר לשפה גנרית במקום להרים אותו לשפת פרימיום.

## פעולה

- הבית הראשי חוזר ל־AppHome, שהוא בסיס חזותי עשיר יותר.
- הגופן מוחלף מ־Arial ל־Heebo דרך google_fonts.
- StudioHome נשאר בקוד כרפרנס בלבד, לא כבית ראשי.

## כיוון עיצוב להמשך

לא חוזרים לבלוקים ענקיים.
כן משתמשים בבסיס העשיר יותר, ואז מכיילים:

- טיפוגרפיה קטנה ומקצועית
- תאורה איכותית
- עומק זכוכית
- כפתורים מדויקים
- מסך מובייל שנראה כמו מוצר חי
