# קו פתוח — Reference Visual System

מטרה:
להפוך את תמונות החזון לשפת מוצר קבועה, כך שכל אתר/מסך שנבנה ייראה קרוב ככל האפשר לשפה שבתמונות.

## עיקרון על

לא מתחילים ממסכים.
מתחילים מיכולות חזותיות.

המסכים הם רק הרכבה של היכולות.

## שפת המוצר

- עיר לילה
- מפה חיה כהה
- נאון ירוק
- כרטיסי זכוכית
- טלפון מרכזי
- קריאות זוהרות
- נהגים כנקודות/פרופילים
- אמון כמדד מרכזי
- דשבורד עשיר
- תחושת תחבורה 2036

## יכולות בסיס

### 1. City Night Backdrop
רקע עירוני כהה עם עומק, קו חוף/אורות, תחושה של מערכת תחבורה חיה.

### 2. Live Network Map
מפה כהה עם גריד, נקודות פעילות, קווי תנועה, קריאה פתוחה, נהגים בסביבה.

### 3. Premium Phone Frame
טלפון שחור מרכזי, עבה, מבריק, עם תאורת נאון סביבו.

### 4. Neon Action Button
כפתור פעולה ירוק גדול, עם זוהר, עומק ותחושת “לחיצה אמיתית”.

### 5. Trust Layer
אמון לא כטקסט קטן אלא כשכבה מוצרית:
מדד אמון, מסמכים נבדקו, נהג מאומת, תגובות לקוחות, נסיעות סגורות.

### 6. Driver Network
נהגים מוצגים כפרופילים חיים:
שם, זמן הגעה, אמון, סטטוס, הצעה.

### 7. Admin Command Center
אדמין נראה כמו חדר בקרה:
קריאות פעילות, נהגים לאישור, אזורים חמים, דוחות, גרפים.

## מסכי רפרנס

### Landing / Poster
מטרה:
למכור את החזון בשנייה אחת.

צריך לכלול:
- שם גדול: קו פתוח
- משפט: רשת קריאות לנהגים ולקוחות
- עיר לילה
- טלפון או רשת קריאות
- שלוש הבטחות: נוסעים באמון, נהגים עצמאיים, לקוחות מחוברים

### Customer
מטרה:
לגרום ללקוח להרגיש שהוא פותח קריאה חיה.

צריך לכלול:
- לאן נוסעים?
- מאיפה אוספים אותך?
- יעד הנסיעה
- עכשיו / מאוחר יותר
- מפה עם מסלול
- כפתור פתח קריאה

### Driver
מטרה:
לגרום לנהג להרגיש שהוא נכנס למרחב עבודה עצמאי.

צריך לכלול:
- אני על הקו
- קריאות פתוחות באזור
- הכנסה היום
- אמון הנהג
- כרטיסי קריאות
- קבל קריאה

### Admin
מטרה:
להראות שיש פה מערכת, לא דף.

צריך לכלול:
- לוח בקרה
- קריאות פעילות
- נהגים לאישור
- דוחות לבדיקה
- אזורים חמים
- פעילות היום

## חוקי ביצוע

1. כל מסך חדש חייב להשתמש ב־ProductShell או בשכבת Shell מקבילה.
2. כל UI חדש חייב להשתמש ב־PremiumPrimitives או להרחיב אותם.
3. אין כפתור רגיל. כל פעולה ראשית היא Neon Action.
4. אין כרטיס רגיל. כל כרטיס הוא Glass / Glow.
5. אין מפה שטוחה. כל מפה היא Live Network.
6. אין “טופס”. יש פעולה מוצרית.
7. אין “דמו”. יש Product Surface.
