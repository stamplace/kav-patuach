# קו פתוח — Asset Performance Pass 01

## בעיה

בלחיצות מהירות בין טאבים או יום/לילה התמונות נראו כאילו הן נתלשות או נטענות באיחור.

## סיבה

Flutter Web מחליף תמונות רקע ומפה בזמן אמת.
כאשר התמונות כבדות, יש רגע שבו התמונה החדשה עדיין לא זמינה.

## תיקון

- נוספה רשימת נכסי פרימיום קבועה.
- כל הנכסים נטענים מראש דרך precacheImage.
- נוספה עטיפת premiumAssetImage עם gaplessPlayback.
- נוספו cacheWidth/cacheHeight כדי להקטין עומס רינדור בדפדפן.
- תמונות הרקע והמפות משתמשות בטעינה חלקה יותר.

## בדיקה

- ללחוץ מהר בין טאבים.
- להחליף יום/לילה כמה פעמים.
- לוודא שאין הבהוב/תלישה משמעותית.
