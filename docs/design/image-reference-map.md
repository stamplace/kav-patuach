# קו פתוח — Image Reference Map

מטרה:
להפוך את תמונות Image2 למפרט פיתוח מחייב עבור אפליקציית Flutter.

## Visual DNA

קו פתוח הוא לא אתר.
קו פתוח הוא אפליקציית תחבורה ויזואלית, כהה, יוקרתית, נאונית, חיה.

## Development Rule

כל מסך Flutter חדש חייב להיבנות כ־Scene.
אין לבנות דף ואז "לעצב אותו".
בונים קודם קומפוזיציה סופית, אחר כך מוסיפים אינטראקציה.

## Definition of Done

מסך נחשב מוכן רק אם:
- בצילום מובייל הוא נראה כמו אפליקציה, לא אתר
- יש פעולה ראשית אחת ברורה
- יש עומק ורקע
- יש נאון/זכוכית/אמון
- אין טקסט מיותר
- הוא קרוב לתמונת הרפרנס המתאימה
