# קו פתוח — Use Case Pass 04: Admin

## מטרה
להפוך את טאב ניהול למסך שליטה ברור.

## Use Case
מנהל רואה:
1. תמונת מצב מערכתית
2. קריאות, נהגים ואמון
3. פעילות היום
4. חריגות שדורשות בדיקה
5. נהגים ודיווחים לאישור

## בדיקה
בטאב ניהול צריך להיות ברור תוך שנייה:
זה לוח שליטה תפעולי, יש מדדים, יש חריגות, ויש פעולות.
