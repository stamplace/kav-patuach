# קו פתוח — Business Scope Checklist

## לפני תחילת עבודה מחייבת

- [ ] התקבל תשלום ראשון
- [ ] הלקוח אישר Scope של 30–60 יום
- [ ] הובהר שזה Pilot ולא Production מלא
- [ ] הובהר ש־Production הוא שלב המשך
- [ ] הובהר שצד ג׳ משולם בנפרד
- [ ] הובהר מה נחשב Done
- [ ] הובהר מה לא כלול
- [ ] נשמרת אפשרות להרחבה לפרודקשן

## סימני סיכון

- [ ] הלקוח מדבר על אלפי משתמשים לפני Scope Production
- [ ] הלקוח מבקש Production בלי תקציב המשך
- [ ] הלקוח מבקש שירותים חיצוניים בלי חשבון/כרטיס שלו
- [ ] הלקוח חושב שהפיילוט הוא מוצר סופי
- [ ] אין אישור כתוב לתכולה
